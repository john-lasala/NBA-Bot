About

NBA-Bot is a bot created in Python that scrapes data from basketball-reference. It current supports 3 commands: PastChampions, CurrentStandings, and LeagueLeaders

NBA-Bot's Commands

Command Description
$pastChampions Gives a list of the past ten champions with their respective year
$currentStandings Shows the current year's standings, user chooses conference
    Here are the options:
  • Eastern Conference
  • Western Conference
$leagueleaders Shows the league leaders for a statistical category, user then chooses which one will be displayed
    Here are the options:
  • Points
  • Rebounds
  • Assists
  • Steals
  • Blocks
  • Free Throw Percentage
